About Me

I'm an Energy Engineering student specializing in Sustainable Energy Engineering, focused on building practical solutions for a cleaner and more efficient energy future. My work combines technical knowledge with creativity — from designing energy systems to developing software tools that bring data and design together.

Alongside my studies, I'm a self taught programmer experienced in web development (HTML, CSS, JavaScript, Node.js), database (MySQL, InfluxDB) and embedded systems using Arduino. I enjoy developing projects that bridge gap between hardware and software, such as sensor-based automation systems and smart energy applications.

I'm also exploring areas like data visualization and IoT integration, aiming to apply technology to real-world sustainability challenges. My long-term goal is to contribute to renewable energy innovation and sustainable infrastructure in Africa.

Smart Home System

This project is a comprehensive home automation system designed to improve security, convenience, and energy efficiency. It uses RFID to automatically unlock doors for authorized tags and integrates a photoresistor to switch lights based on ambient light.

Built on an Arduino platform, the project demonstrates hands-on experience in programming, sensor interfacing, and embedded system design.

Environmental Monitoring system

This Arduino-based environmental monitoring system provides continuous real-time monitoring of temperature and humidity, useful for healthcare environments.

It uses a DHT11 sensor and displays data on an I2C LCD; a buzzer alerts when temperature crosses 37°C.

2-Octave Keypad Synth

The 2-Octave Keypad Synth is an interactive Arduino-based project that transforms a 4x4 keypad into an instrument capable of playing two octaves.

Music Streaming Web App

An in-progress music streaming web app built with Node.js, Express, and MySQL. Supports sign up, login, browsing songs and recently played tracking.
